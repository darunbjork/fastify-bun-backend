# bun-react-template

To install dependencies:

```bash
bun install
```

To start a development server:

```bash
bun run dev

```

To run for production:

```bash
bun start
```

To run tests
```bash
bun test
```



This project was created using `bun init` in bun v1.3.6. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
